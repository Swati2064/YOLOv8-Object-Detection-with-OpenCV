# 🚀 YOLOv8 Object Detection with Python & OpenCV

A Computer Vision project that demonstrates real-time object detection using the YOLOv8 model with Python and OpenCV. This repository includes image detection, video detection, webcam object detection, and object counting.

---

## 📌 Features

- ✅ Object Detection on Images
- ✅ Real-Time Webcam Detection
- ✅ Video Object Detection
- ✅ Object Counting
- ✅ YOLOv8 + OpenCV Integration
- ✅ COCO Dataset Classes Support
- ✅ Bounding Boxes with Confidence Scores

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/YOLOv8-Object-Detection-with-OpenCV.git
```

Move into the project folder

```bash
cd YOLOv8-Object-Detection-with-OpenCV
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## 📥 Download YOLOv8 Model

Download the pretrained model:

```
yolov8n.pt
```

Place it inside the project folder.

---

## ▶️ Run the Project

### Image Detection

```bash
python 2.yolov8_basics.py
```

### Video Detection

```bash
python 3.yolov8_n_opencv.py
```

### Webcam Detection

```bash
python customer_detection.py
```

### Object Counting

```bash
python object_count.py
```
